Projects

Scratch

This assignment was the first pair-programming project done in my CSP class. The code was done on a beginner level coding site called Scratch, a free online programming language developed by MIT. Unlike normal coding languages, Scratch uses a blocks. These blocks contain codes and you snap them together to create lines of code. This is a great site for beginner coders because it is easy and intuitive. Scratch is good for making small games and animations.

MIT App Inventor

This was another pair-programming project. The code was programmed on another MIT website called MIT App Inventor. The language is very similar to Scratch. App Inventor also uses blocks, but different types. Like Scratch, App Inventor specializes in making apps. Depending the creator's decision, the app can become published and people may even use it! This is a great site to test some prototypes for your app ideas.
